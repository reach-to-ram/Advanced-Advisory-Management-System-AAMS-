Published: June 15, 2023 • 09:45 AM

New Malware Targeting Defence Networks Discovered

Security researchers have identified a sophisticated malware campaign specifically targeting defence networks across multiple countries. The malware, dubbed "ShadowStrike," appears to be part of a coordinated cyber espionage operation.

Key Findings:

  • The malware uses advanced obfuscation techniques to evade traditional detection methods
  • Initial infection vectors include spear-phishing emails targeting specific personnel
  • Once deployed, it establishes encrypted communication channels with command and control servers
  • Data exfiltration capabilities focus on classified documents and network configuration
  • Evidence suggests the campaign has been active for approximately 3 months

Affected Systems:

The malware primarily targets Windows-based systems within secure networks, with specific focus on workstations with access to classified information management systems.

Recommended Actions:

  1. Immediately update all endpoint protection solutions with the latest signatures
  2. Conduct a full system scan using the provided IOCs (Indicators of Compromise)
  3. Implement network traffic analysis to identify suspicious communication patterns
  4. Enforce multi-factor authentication for all remote access
  5. Review and restrict privileged account access

CERT-In is actively monitoring this threat and will provide additional updates as more information becomes available. Defence organizations are advised to implement the recommended mitigations with highest priority.
